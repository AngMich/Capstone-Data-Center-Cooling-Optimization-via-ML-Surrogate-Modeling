**Capstone: Data Center Cooling Optimization via ML Surrogate Modeling
**
This capstone project develops a machine learning surrogate model for energy efficiency estimation and operational scenario analysis in data center cooling systems. A complete data science pipeline — encompassing data cleaning, feature engineering, model selection, hyperparameter tuning via GridSearchCV, and deployment — was implemented on a publicly available data center dataset. Four regression models were evaluated (Ridge, BayesianRidge, XGBoost, HistGradientBoostingRegressor, and a neural network via skorch); Ridge regression was selected for cooling power prediction and quantile regression for outlet temperature prediction. The trained models are deployed through an interactive Streamlit dashboard hosted on Hugging Face, accepting four user-defined operational inputs and returning Power Usage Effectiveness (PUE), estimated operational cost, and thermal stability metrics, enabling what-if scenario analysis without intervention on live infrastructure.
